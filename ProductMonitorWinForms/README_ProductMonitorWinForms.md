# ProductMonitorWinForms

Aplicativo simples em WinForms para monitorar consultas de produtos. Exibe uma tabela com a quantidade de pesquisas por produto e possui um ícone dinâmico na bandeja do sistema.

## Compilação e Execução

1. Certifique-se de ter o .NET SDK 6.0 ou superior instalado.
2. Navegue até a pasta `ProductMonitorWinForms`.
3. Execute:

```bash
 dotnet run
```

O aplicativo será iniciado com o ícone na área de notificação. Utilize o menu do ícone para abrir o painel, forçar reconexão (simulada) ou sair do aplicativo.
